Dragon Sculpt

One day I came across an event in a Discord group about doing a dragon sculpting in a week and decided to do the sculpting with my own efforts. I challenged myself to create this sculpture based on single image.

Thanks to the community specializing in 3D graphics, I was able to learn a lot of things and get to know the working environment more broadly.

  • Importing terrain from ASCII XYZ file
  • Importing buildings from GML file
  • Selecting and removing objects
  • Mesh simplification (flattening roofs, creating bounding boxes)
  • Exporting the 3D scene as an OBJ file
