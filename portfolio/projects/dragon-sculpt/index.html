Dragon Sculpt

One day I came across an event in a Discord group about doing a dragon sculpting in a week and decided to do the sculpting with my own efforts. I challenged myself to create this sculpture based on single image from ArtStation.

Thanks to the community specializing in 3D graphics, I was able to learn a lot of things and get to know the working environment more broadly.
