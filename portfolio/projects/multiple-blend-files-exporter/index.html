Multiple blend files exporter

The purpose of this project

Before starting game development, we decided to create a test level using assets from other projects. One of our graphic designers had a Blender (.blend) file containing various models and assets in a single file. Since Blender uses Python scripting, I considered writing a script that could export all the models with a single click, streamlining the workflow and speeding up level design. The script would automatically detect each model in the .blend file and export them as separate GLB files with textures, ready for use in the game engine.

How it works

  • Set the input path - specify the folder containing one or more blend files in the filepath variable.
  • Define the output path - enter the destination folder for exported .glb models in the filepath_output variable.

Once the paths are correctly set, the script will:

  • Process all blend files within the specified input folder.
  • Export the results as .glb files to the designated output directory.

Blender file: gltf_exporter.blend

Tools used: Python, Blender
