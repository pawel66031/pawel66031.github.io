Building Generator

There was an opinion about a new feature in Blender called Geometry Nodes. Most graphic designers had the opinion that the new tool is pointless and can't do something decent. Since I have knowledge of creating tools from scratch and experience with a program for creating procedural objects, namely Houdini, I decided to prove to people that Geometry Nodes has great possibilities and can save a lot of time on modeling.

When Blender introduced Geometry Nodes, many graphic designers initially dismissed the feature as pointless, believing it couldn't produce high-quality results. Having experience in tool development and procedural modelling (particularly with Houdini) I saw an opportunity to challenge this perception. My goal was to demonstrate that Geometry Nodes offer powerful capabilities and can significantly speed up the modelling process.

This project revolves around a procedural building generator that utilizes drawn lines and a flat mesh (plane) as input. Due to Blender's current limitations in the Modifiers section - such as the inability to directly use Empty objects or list with multiple objects. I had to find an alternative way to define openings in walls. The only viable solution in Blender 4.3 was to use an additional Geometry Nodes modifier to specify the position and size of these openings.

The result is an efficient tool that generates buildings instantly when applied to a flat curve or plane.

Results

  • Adjustable wall thickness
  • Procedural window and door placement using a separate Geometry Nodes modifier
  • Customizable roof properties, including rotation, position and height
