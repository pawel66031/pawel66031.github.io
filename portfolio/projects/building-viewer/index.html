BuildingViewer

The motivation behind creating this program was to find an alternative way to import terrain and building data from the geoportal.gov.pl website, rather than relying on OpenStreetMap for 3D map data. The Polish geoportal provides more detailed data compared to OpenStreetMap, making it a valuable resource for precise mapping.

BuildingViewer is specifically designed for the geoportal.gov.pl website, allowing users to easily import data, edit objects and export them back as an OBJ file compatible with many 3D modelling software applications. This program includes features such as:

  • Importing terrain from ASCII XYZ file
  • Importing buildings from GML file
  • Selecting and removing objects
  • Mesh simplification (flattening roofs, creating bounding boxes)
  • Exporting the 3D scene as an OBJ file
